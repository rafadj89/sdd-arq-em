# /specs — Documentación Técnica del Proyecto

Directorio de especificaciones técnicas y contexto regulatorio para la plataforma de interoperabilidad de electromovilidad UPME.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `resolucion-40559-contexto.md` | Contexto de la Resolución 40559 de 2025, plazos regulatorios, actores, módulos OCPI requeridos |
| `guia-integracion-cpo.md` | Manual de integración para CPOs: registro en Cárgame, onboarding, seguridad, rate limits, SLA |
| `stack-tecnologico.md` | Stack tecnológico completo: backend, frontend, data, infra, seguridad, ambientes |
| `areas-colaboracion.md` | Áreas de colaboración externas al equipo IT: Legal, Seguridad, Entidad Externa, CPOs |
| `arquitectura-c4-ddd.md` | **Arquitectura C4 + DDD completa:** Context Map, 4 niveles C4 (System Context, Containers, Components, Code), Domain Models, Domain Events, Kafka Topics, Flujos de Secuencia, ADRs, Deployment OCI |

## Nota sobre la Resolución 40559

Los plazos exactos de la resolución deben ser validados contra el documento oficial publicado por el Ministerio de Minas y Energía. Los plazos en estos documentos son estimaciones basadas en el contexto regulatorio colombiano de electromovilidad y deben actualizarse cuando se tenga acceso al texto definitivo.

**Fuentes oficiales a consultar:**
- MinEnergía: https://www.minenergia.gov.co
- UPME: https://www1.upme.gov.co
- Diario Oficial: https://www.suin-juriscol.gov.co
